# Study Planner Frontend

Frontend da aplicação Study Planner Pro desenvolvido com React e Tailwind CSS.

## Tecnologias Utilizadas

- React 19.2.0
- Vite 7.2.4
- Tailwind CSS 4.1.17
- React Router DOM 7.9.6
- Axios 1.13.2
- Recharts 3.5.1
- Lucide React (ícones)

## Como Executar

```bash
# Instalar dependências
pnpm install

# Executar em desenvolvimento
pnpm dev

# Build para produção
pnpm build
```

A aplicação estará disponível em: `http://localhost:5173`

## Configuração da API

O frontend está configurado para se conectar ao backend em `http://localhost:8080/api`.

Para alterar, edite `src/services/api.js`.
